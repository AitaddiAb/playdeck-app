### Planned Components

#### Core Components
- **AppPage**: Unified page wrapper component
- **AppDialog**: Reusable dialog component for forms and confirmations
- **AppHeader**: Application header with navigation
- **AppSidebar**: Sidebar navigation component
- **AppFooter**: Mobile footer navigation

#### Game Components
- **GameCard**: Individual game card display
- **GameGrid**: Grid layout for game library
- **GameList**: List layout for game library
- **GameDetails**: Game detail view/modal
- **GameSearch**: Search component for games

#### Settings Components
- **SettingsSection**: Settings category section
- **SettingsToggle**: Toggle switch for settings
- **SettingsInput**: Input field for settings
