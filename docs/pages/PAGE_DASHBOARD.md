## Description

Main dashboard/home page for the Playdeck application. Currently displays a placeholder message and serves as the entry point for the game library overview.

## Components Used

Currently, the page uses basic HTML elements. Future components may include:

- Game library grid/list
- Statistics cards
- Recent games section
- Quick actions

## Features

- Route name: `Dashboard`
- Lazy-loaded component for code splitting
- Placeholder content ready for implementation
